4. PAYPAL INTEGRATION (SMART BUTTONS)
---------------------------------------

Use PayPal JavaScript SDK (popup experience, NOT redirect page)

Environment variables:
- PAYPAL_CLIENT_ID
- PAYPAL_CLIENT_SECRET
- PAYPAL_WEBHOOK_ID
- PAYPAL_ENV=sandbox

Backend:

Create:
POST /api/paypal/create-premium-order

- Auth required
- Owner only
- Create PayPal order (intent: CAPTURE)
- Include:
  custom_id = user.id
  invoice_id = premium_<userId>_<timestamp>
- Save pending purchase
- Return:
  orderID

Webhook:
POST /api/paypal/webhook

- Verify signature
- Handle:
  PAYMENT.CAPTURE.COMPLETED
  PAYMENT.CAPTURE.PENDING
  PAYMENT.CAPTURE.DENIED

On completed:
- update purchase
- set user.isPremium = true
- set premiumActivatedAt
- set premiumSource = "paypal"

Make webhook idempotent
